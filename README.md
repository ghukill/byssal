# byssal
Byssal - A Repository of Threads

## Development

### Installation

1- Clone repository
```shell
git clone https://github.com/ghukill/byssal
cd byssal
```

2- Create virtual environment and install dependencies:
```shell
uv venv .venv
uv sync
```

3- Run tests:
```shell
uv run pytest -vv
```

3- Run linting:
```shell
uv run ruff check src/
```

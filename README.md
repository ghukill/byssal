# byssal
Byssal - A Repository of Threads

## Development

### Installation

1- Clone repository

2- Create virtual environment and install dependencies:
```shell
uv venv .venv
uv sync
```

## Deploy Byssal Repository

1- Create directory
```shell
mkdir my-byssal
cd my-byssal
```

2- Create virtual environment
```shell
uv 
```
